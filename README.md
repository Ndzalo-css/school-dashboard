🚀 School Dashboard DevOps Project

A real-world full-stack DevOps project that demonstrates how to build, containerize, deploy, and monitor a modern application.
🧠 Project Overview

This project simulates a student performance dashboard used by educators to monitor student progress and identify at-risk learners.

It includes:

📊 Interactive frontend dashboard
🔗 Backend API serving real data
🐳 Dockerized services
☁️ Cloud deployment (Render + GitHub Pages)
📈 Monitoring with Prometheus & Grafana

Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Containerization: Docker
CI/CD: GitHub Actions
Monitoring:
Prometheus
Grafana
